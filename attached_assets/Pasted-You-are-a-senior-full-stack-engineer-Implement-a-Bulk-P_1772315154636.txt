FEATURE: ADMIN BULK PRICE MANAGER

1) Admin UI

  - step=0.01

  - > 0
  - max 2 decimals

- Optional: also set compare-at price

B) Increase/decrease by percent
- +10%, -15%

  - default: end in .00 (configurable)
  - optional: end in .99 (toggle)

C) Increase/decrease by fixed amount
- +$5.00 / -$3.00

D) Apply “Sale” pricing
- Set compare-at to current price
- Apply discount percent to price
- Example: compare-at $49, price $39

E) Revert last bulk change

bulk-price-update payload example:
{
  productIds: [...],
  operation: "set" | "percent" | "fixed" | "sale",
  priceDollars?: number,
  compareAtDollars?: number | null,
  percent?: number,
  amountDollars?: number,

- verify admin auth
