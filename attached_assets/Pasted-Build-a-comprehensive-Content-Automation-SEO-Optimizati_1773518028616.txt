Build a comprehensive **Content Automation, SEO Optimization, and Site Integrity System** for the NurseNest platform.

This system must ensure the platform continuously grows high-quality educational content while maintaining strong search engine visibility and preventing broken pages or incomplete content.

The system should operate across the entire NurseNest ecosystem including:

• Nursing tiers (RPN, RN, NP)
• Allied Health programs
• New Grad support resources
• Lessons
• Test bank questions
• Flashcards
• Exams
• Mock exams
• Blog articles
• Career resources

The system must include automated content generation, SEO optimization, coverage analysis, and site health monitoring.

PRIMARY OBJECTIVES

1. Improve SEO discoverability for Allied Health pages.
2. Improve SEO discoverability for New Grad pages.
3. Automatically generate missing lessons and educational content.
4. Automatically generate exam questions and flashcards where coverage is low.
5. Detect and repair broken links and missing pages.
6. Create long-form SEO content that ranks in Google.
7. Build a structured internal linking system.
8. Maintain site integrity so the platform never advertises content that does not exist.

PROGRAM-LEVEL SEO LANDING PAGES

Create high-authority landing pages for each healthcare program.

These pages should target major search queries related to certification exam preparation.

Examples:

/respiratory-therapy-exam-prep
/paramedic-exam-prep
/medical-lab-tech-exam-prep
/diagnostic-imaging-exam-prep
/occupational-therapy-exam-prep
/physical-therapy-exam-prep

• profession overview
• certification exam explanation
• required clinical knowledge
• exam difficulty discussion
• study strategies
• links to flashcards
• links to practice questions
• links to lessons
• links to blog articles

Each page must contain **1500 to 2500 words of educational content**.

These pages should act as SEO entry points for search traffic.

CONTENT COVERAGE ANALYZER

Create an admin dashboard tool:

Admin → Content Coverage

The system should analyze:

• number of questions per tier
• number of questions per specialty
• number of questions per topic
• number of flashcards per topic

Default coverage targets:

Nursing tiers:

• minimum 300 questions per major system
• minimum 500 questions per specialty

Allied health programs:

• minimum 300 questions per profession

Flashcards:

• minimum 25 flashcards per topic

If coverage is below targets, automatically generate additional questions or flashcards.
