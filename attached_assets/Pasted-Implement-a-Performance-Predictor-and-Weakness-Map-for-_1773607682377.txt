DATA INPUT

Use existing user data:

-----------------------------------

PASS PROBABILITY CALCULATION

Create a pass probability score.

Inputs

Accuracy
Weighted by question difficulty

Recent performance
Last 200 questions weighted higher

Mock exam scores

Adaptive CAT difficulty

Example algorithm

passProbability =
(accuracy * 0.5) +
(mockExamAverage * 0.3) +
(catDifficultyScore * 0.2)

Output

Display as percentage

Example

"78% Likely to Pass"
