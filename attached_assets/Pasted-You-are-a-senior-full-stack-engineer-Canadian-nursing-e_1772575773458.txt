CAT MODE (CORE FEATURE)
Implement a CAT-like practice mode (not claiming to exactly replicate official algorithm; it is a learning simulator):
- Session config: 90–150 questions, 4 hours max timer (optional), user can choose:
  - Full CAT (adaptive)
  - Mini CAT (e.g., 25, 50)
- Adaptive logic (MVP):
  - Maintain estimated ability theta per domain (or global)
  - Choose next question:
    - prioritize domains using target blueprint weights vs current domain counts
    - within chosen domain, choose difficulty near user’s current theta
    - avoid repeats using questionIds seen
  - After each answer, update theta:
    - correct => + step scaled by difficulty
    - incorrect => - step scaled by difficulty
  - Keep an “uncertainty” measure; if uncertainty low and minQ reached, allow early stop only in Mini CAT; for Full CAT keep 90–150 bounds.
- Provide a clear UI disclaimer: “CAT-like training simulator” and explain it adapts difficulty; not a guarantee of pass/fail.
