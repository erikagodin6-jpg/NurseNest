## Regional / Exam Architecture

### Region‑Aware Exam Structure

```
You are a senior Next.js architect and learning platform engineer.

Implement region-aware exam architecture in NurseNest.

CONTEXT:
NurseNest serves BOTH United States and Canadian nursing users.

Requirements:

1) REGION-SPECIFIC EXAM STRUCTURE

If site context = United States:
- Display US exam frameworks
- Include NCLEX-RN, NCLEX-PN, and US NP population-focused exams
- Use US clinical conventions and terminology

If site context = Canada:
- Display Canadian exam frameworks
- Include REX-PN, NCLEX-RN (Canada), and Canadian NP exams (CNPE streams)
- Use Canadian lab units and conventions

2) CONTENT ISOLATION
- Exams, lessons, and question banks must render conditionally by region
- Avoid duplication of pages
- Region context must drive terminology, lab units, and exam labeling

3) IMPLEMENTATION
Provide:
- Region detection strategy
- Exam configuration model
- Conditional rendering logic
- Safe routing structure
```
