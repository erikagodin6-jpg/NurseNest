Create a structured “Education Pathway for Success” section on NurseNest that guides students step-by-step based on their level and goals.

Goal:
Help users self-identify where they are and automatically route them to the correct learning pathway.

Do NOT redesign the whole site.
Add a new structured section (or page) called:

“Choose Your Path to Success”

=========================================
PART 1 — USER SELF-SELECTION OPTIONS
=========================================

Create four clear pathway cards:

1) Pre-Nursing / Foundations
   - “I’m preparing for nursing school.”
   - Route: Pre-Nursing Foundations modules

2) RPN / LPN (REx-PN / NCLEX-PN)
   - “I’m studying for my practical nurse exam.”
   - Route: RPN lessons + REx-PN / NCLEX-PN prep

3) RN (NCLEX-RN / Canadian RN)
   - “I’m preparing for my RN exam.”
   - Route: RN system modules + clinical judgment + delegation

4) NP (CNPE / CNPLE Coming Soon / AANP / ANCC)
   - “I’m preparing for NP practice or exams.”
   - Route: Advanced diagnostics + prescribing + referral modules

Each card must include:
- Short description of what the pathway includes
- Skills they will master
- Link to start pathway
- Optional “See full curriculum” link
