Implement a flashcard preview system for NurseNest that allows users to try the flashcard experience before upgrading.

Flashcards are premium content, but users must be able to experience the interface and see the value before hitting the paywall.

FREE PREVIEW RULES

Free users should be able to:

• access a limited flashcard preview session
• answer a small number of cards
• see sample rationales
• view the interface and study flow

Limit preview to:

3–5 flashcards per session
or
10 flashcards per day

After the preview limit is reached, display an upgrade screen.

PREVIEW SESSION EXPERIENCE

The preview session should behave like the real flashcard system.

User should see:

• question
• answer choices
• rationale
• incorrect option explanations
• clinical pearl
• related lesson link

However:

• progress is not permanently saved
• advanced modes are locked
• adaptive review is disabled

PAYWALL TRIGGER

After the preview limit is reached show a premium upgrade screen.

Headline:
Unlock the Full Flashcard Library

Text:
Continue practicing unlimited exam-style flashcards with full rationales, adaptive review, and lesson-linked explanations.

Premium features list:

• unlimited flashcards
• all nursing tiers
• weak areas mode
• adaptive review
• saved progress
• flagged and mastered cards
• lesson-linked remediation

Buttons:

Upgrade Now
View Plans

Optional secondary option:

Continue with Limited Preview Tomorrow

UX REQUIREMENTS

• preview should feel like real study
• upgrade prompt should appear after value is demonstrated
• avoid aggressive paywall before users see flashcard interface
• make premium benefits clear

TECH IMPLEMENTATION

Add logic:

if user.plan === "free":
   allow previewCardsRemaining
   decrement per card

if previewCardsRemaining === 0:
   redirect to upgrade modal

if user.plan === "premium":
   unlimited flashcard access

ADMIN SETTINGS

Admin should be able to configure:

• preview card limit
• preview topics
• preview tiers
• upgrade message text

ACCEPTANCE CRITERIA

• free users can experience flashcards
• premium users have unlimited access
• paywall appears after preview limit
• preview shows real flashcard experience
• upgrade prompt highlights premium value
