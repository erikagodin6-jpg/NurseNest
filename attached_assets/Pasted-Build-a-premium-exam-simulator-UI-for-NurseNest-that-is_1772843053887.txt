Build a premium exam simulator UI for NurseNest that is strongly inspired by real licensure exam workflows and high-end medical Qbank platforms, but is NOT a direct copy of UWorld, Pearson Vue, NCLEX, or any copyrighted interface. It must feel familiar to nursing students, clinically professional, calm, modern, feminine-soft, and fully aligned with the NurseNest brand aesthetic.

GOAL
Create a realistic exam-taking experience that feels like a real nursing licensure testing environment, while preserving NurseNest branding, polish, and usability across desktop, tablet, and mobile.

CORE DESIGN DIRECTION
The interface should feel:
- exam-authentic
- clean and distraction-free
- modern clinical
- premium educational SaaS
- soft, polished, and trustworthy
- feminine but not childish
- optimized for long study sessions and reduced cognitive overload

DO NOT:
- copy any copyrighted platform exactly
- reuse another company’s icon layout, wording, exact spacing, or exact component arrangement
- use harsh neon colors
- make it look cartoonish, playful, or overly “cute”

INSTEAD:
- create an original NurseNest exam UI system that borrows the best UX principles of real exam platforms:
  - focused question area
  - clear answer selection
  - top status/navigation bar
  - side or bottom tools area
  - timer visibility
  - flag/mark for review
  - previous/next navigation
  - exam progress visibility
  - rationale mode after submission
  - image/table/media support
  - lab values and reference popups
  - strikeout/highlight/noteboard/calculator functions
  - confidence-building structure and pacing

BRAND SYSTEM
Use NurseNest brand styling throughout:
- rounded corners but not overly rounded
- soft shadows
- premium card surfaces
- subtle gradients only where appropriate
- elegant whitespace
- clean typography
- soft pastel clinical palette
- high contrast for readability
- gentle brand accents for active states, selected answers, progress, toggles, and buttons

Create 4 interchangeable NurseNest exam themes:
1. Clinical Light
   - ivory or soft white background
   - cool gray text
   - dusty lavender accents
   - soft blue secondary accents
2. Lavender Dream
   - pale lavender surfaces
   - muted plum text
   - lilac active states
   - very soft pink highlights
3. Pastel Lilac
   - white base
   - pastel lilac primary
   - powder blue secondary
   - sage for success states
4. Calm Night
   - dark muted navy-lilac background
   - off-white text
   - lavender and dusty blue accents
   - low-glare dark mode for long study sessions

TYPOGRAPHY
Use a modern sans-serif stack that feels professional and soft.
Prioritize readability over decorative styling.
Recommended style:
- headings: modern, clean, slightly rounded
- body: highly legible exam-reading font
- strong hierarchy for stem, answer options, rationale headings, table headings, and tool labels

Create a typography system with:
- exam header labels
- question stem text
