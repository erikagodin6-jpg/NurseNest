PART 4: TUTOR MODE VS EXAM MODE

Implement a clear distinction between Tutor Mode and Exam Mode.

Tutor Mode:
- rationale shown after each question
- images shown in rationale
- answer review immediately visible
- optional confidence rating after answer
- allow note-taking and highlighting if those features already exist or can be added later

Exam Mode:
- no rationale until the end
- timed experience
- lock answer after submission
- test navigation follows exam settings
- final score and review page at the end
- cleaner, more focused exam-taking view

The mode selector should appear before quiz start and also be visible in the study header during the session.
