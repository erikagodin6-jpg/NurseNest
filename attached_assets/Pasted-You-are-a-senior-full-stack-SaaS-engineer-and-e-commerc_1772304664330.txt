Build a fully functional Digital Study Marketplace inside NurseNest to sell:

- Nursing cram booklets (PDF)
- Study guides

====================================================
GOAL
====================================================

Create a clean, simple, revenue-focused marketplace that allows:

2) Users to securely purchase and download
3) Stripe payment integration
4) Automatic access delivery
5) Sales tracking
6) Coupon code support

====================================================
PHASE 1 — DATABASE STRUCTURE
====================================================

Create table: digital_products

Fields:
- id
- slug
- title
- description
- shortDescription
- price
- compareAtPrice (optional)
- fileUrl (secure storage link)
- coverImageUrl
- category (Cram Guide, Flashcard Pack, Printable, Bundle)
- tierTarget (RPN/RN/NP/All)
- examTarget (REx-PN, NCLEX, NP)
- featured (boolean)
- createdAt
- updatedAt
- isActive

Create table: product_purchases

Fields:
- id
- userId
- productId
- stripeSessionId
- purchaseDate
- downloadCount
- maxDownloads (default 5)
