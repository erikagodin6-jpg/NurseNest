========================================================

- largest heading block
- or a large above-the-fold card/grid

Then implement targeted fixes depending on what LCP is:

IF LCP IS AN IMAGE:
- Convert to WebP (or AVIF if supported) and reduce size to <200KB.
- Ensure width/height are set to prevent layout shifts.
