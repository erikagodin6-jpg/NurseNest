STEP 4: BATCHED GENERATION LOOP (MANDATORY)

Initialize:
target = questionCount

attempts = 0
maxAttempts = 20

    remaining = target - generated.length

    Parse JSON strictly
    Validate each question
    Remove duplicates (hash normalized stem)
    Append valid questions
    attempts++

If after maxAttempts generated.length < target:
    Throw structured error:

      requested: target,
      attempts: attempts,
