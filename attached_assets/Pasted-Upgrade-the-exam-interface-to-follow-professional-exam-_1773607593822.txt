FEATURE 1 — STUDY MODE VS EXAM MODE

Add a toggle at the top of the exam screen.

Study Mode
• answer shows immediately
• rationale panel opens automatically
• flashcard style learning

Exam Mode
• answer hidden until user clicks "Reveal Answer"
• rationale panel hidden until reveal
• simulates real exam conditions

Implementation
Add a toggle switch component:

Study Mode | Exam Mode

Default: Study Mode
