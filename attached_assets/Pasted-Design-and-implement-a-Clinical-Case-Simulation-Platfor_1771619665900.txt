CRITICAL SAFETY CONSTRAINTS

- AI must NEVER autonomously modify clinical content
- AI must NOT invent or assume guideline updates
- All updates must originate from authoritative sources
- The system must support verification workflows
- The system must protect against outdated or incorrect medical guidance

SYSTEM OBJECTIVES

Create a conceptual design for a Guideline Monitoring & Update Assistance System with the following properties:

1) Source-Anchored Monitoring

- Track updates only from authoritative bodies

United States examples:
- CDC
- NIH
- FDA
- USPSTF
- Specialty societies (ACC, AHA, etc.)

Canada examples:
- Health Canada
- Public Health Agency of Canada (PHAC)
- Canadian Cardiovascular Society (CCS)
- CNA / specialty organizations

2) Change Detection Logic

- Detect when source documents change or update
- Identify version differences
- Flag potentially affected NurseNest content

3) Verification-First Workflow

- AI proposes POSSIBLE update areas
- Human review required before publication
- No automatic publishing allowed

4) Jurisdiction Separation

- Ensure Canadian site references Canadian guidance
- Ensure US site references US guidance
- Prevent cross-jurisdiction contamination

5) Auditability & Traceability

- Track when content was last reviewed
- Store source references
- Maintain revision history

6) Drift Prevention Mechanisms

- Alert when content ages beyond threshold
- Flag high-risk topics (drugs, protocols, dosing, emergencies)

7) AI Role Limitations

AI may:
- Summarize detected changes
- Compare document versions
- Identify impacted concepts

AI may NOT:
- Assert clinical correctness
- Generate new medical rules
- Replace human clinical oversight

- System architecture overview
- Data flow model
- Update detection strategy

- Example lifecycle of a guideline update

PAGE STRUCTURE REQUIREMENTS
------------------------------------------------

1. System Overview

2. Cellular & Functional Foundations (MANDATORY)

3. Core Physiological Mechanisms
- Stepwise causal explanations
- Include mechanism maps (A → B → C → D)
- Include compensatory mechanisms if relevant

4. Clinical Correlation Sections
Use titled blocks:

• Why This Matters in Real Patients
• Clinical Pattern Recognition
• Common Misconceptions
• Clinical Pearls
