Implement a theme-aware branding system so the NurseNest logo, top header bars, and navigation menus automatically match the theme a user selects.

Goal
When a user switches themes (for example Pastel Lilac, Lavender Dream, Clinical Light, Dark Mode), the logo color accents and UI bars automatically adapt to that theme without breaking layout or branding.

Key rules

1. Do not redesign the logo artwork.
2. The leaf icon may change color based on the theme.
3. The wordmark should remain neutral (dark or white depending on background).
4. The logo must remain crisp SVG.
5. Do not change spacing between the leaf and “NurseNest”.

Implementation

1. Use the SVG logo as a theme-aware component.

Convert the leaf color in the SVG to use `currentColor` so it inherits the theme color.

Example:

<svg viewBox="0 0 300 80" class="logo">
  <path class="logo-leaf" fill="currentColor" ... />
  <path class="logo-text" fill="var(--logo-text)" ... />
</svg>

2. Create theme design tokens.

Add CSS variables that control all theme colors.

Example root variables:

:root {
--brand-primary: #bfa8ff;
--brand-secondary: #d8ccff;
--logo-text: #1f2937;
--nav-bg: #ffffff;
--nav-border: #ececf3;
}

3. Define theme palettes.

Example themes:

Pastel Lilac
[data-theme="lilac"] {
--brand-primary: #bfa8ff;
--brand-secondary: #e7deff;
--nav-bg: #ffffff;
}

Lavender Dream
[data-theme="lavender"] {
--brand-primary: #c7b6ff;
--brand-secondary: #f0ebff;
}

Clinical Light
[data-theme="clinical"] {
--brand-primary: #5ea7ff;
--brand-secondary: #e7f1ff;
}

Dark Mode
[data-theme="dark"] {
--brand-primary: #9ca8ff;
--nav-bg: #0f172a;
--logo-text: #ffffff;
}

4. Apply theme colors to UI elements.

Header bars:

.header-bar {
background: var(--brand-primary);
}

Navbar:

.navbar {
background: var(--nav-bg);
border-bottom: 1px solid var(--nav-border);
}

Menu hover states:

.nav-link:hover {
color: var(--brand-primary);
}

Buttons:

.btn-primary {
background: var(--brand-primary);
}

5. Make the logo inherit the theme color.

.logo {
color: var(--brand-primary);
}

This automatically colors the leaf icon based on the selected theme.

6. Persist theme choice.

Store theme selection in localStorage so it stays when the user returns.

Example:

localStorage.setItem("theme", selectedTheme)

Load it on page startup.

7. Ensure the system works across the entire site.

Confirm theme styling applies to:

* main NurseNest site
* allied health subdomain
* new grad pages
* dashboards
* admin panels
* mobile navigation

8. Do not break layout.

Do not change:

* navbar height
* logo spacing
* menu alignment
* responsive behavior

9. Validate

Test theme switching on:

* desktop
* mobile
* logged-in dashboards
* marketing pages
* pricing pages

Ensure the logo color and navigation bars update instantly when the theme changes.

End result

Users can select a theme and the following will automatically match:

• NurseNest leaf logo color
• top announcement bars
• navigation background
• menu hover colors
• buttons and UI accents
