Create an automated Admin Content Metrics Dashboard for NurseNest that displays the current number of questions per tier, total question counts, growth projections, and generation pipeline status. This dashboard must be visible only to admin accounts and update dynamically from the production database.

Do not create mock data. Pull real values from the production tables that store exam questions and generation tasks.

Create a new admin route:

This page should display clean analytics cards similar to modern SaaS dashboards.

Use the existing NurseNest design system, DM Sans typography, soft rounded cards, and the current pastel theme tokens.

Cards should have:

soft shadow
rounded corners
large numeric metric
small description text underneath

Section 2 — Query Production Database

Write backend logic to pull counts from the production question tables.

Group counts by tier.

Required tiers:

RN (NCLEX-RN)
RPN (REx-PN / LVN)
NP (Nurse Practitioner)
RRT (Respiratory Therapy)
Emergency Nursing (Allied)

Return counts of all published questions only.

Exclude drafts and unpublished items.

Example logic:

SELECT tier, COUNT(*) 
FROM exam_questions
WHERE status = 'published'
GROUP BY tier

Return results to the dashboard API.
