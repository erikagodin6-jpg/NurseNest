MASTER REPLIT PROMPT
Implement NCLEX-Style Case Study Engine for NurseNest
Build a clinical case study exam system for NurseNest that allows exam creators (admin) to create multi-question clinical scenarios similar to NCLEX clinical judgment questions.
The system must support RPN, RN, and NP tiers and different difficulty levels.
 
CORE CONCEPT
A case study is a patient scenario followed by 4 to 6 linked questions.
The scenario evolves as new information becomes available.
Example progression
Step 1
Patient presentation
Step 2
Assessment findings
Step 3
Lab results or imaging
Step 4
Intervention
Step 5
Evaluation
Each step unlocks new questions.
 
DATABASE STRUCTURE
Create a new table
case_studies
Fields
id
title
tier (rpn, rn, np)
difficulty (easy, medium, hard)
system (cardiac, respiratory, endocrine, maternity, mental health etc)
scenario_intro
created_at
 
Create another table
case_study_steps
Fields
id
case_id
step_number
clinical_update_text
exhibit_data_json
