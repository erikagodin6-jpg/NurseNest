1) Admin shop CRUD (edit/delete/price in dollars/admin download)
Data model requirements (minimum)

products.price_cents (int) stays canonical

Add products.is_deleted (bool default false)

Add products.updated_at

(Optional) products.status = draft|published|archived

Admin API (must exist)

GET /api/admin/products?includeDeleted=0|1

POST /api/admin/products

PUT /api/admin/products/:id

DELETE /api/admin/products/:id (soft delete: set is_deleted=true)

POST /api/admin/products/:id/restore

POST /api/admin/products/:id/price body: { "priceDollars": 19.99, "compareAtDollars": 29.99 }

server converts to cents with correct rounding:

priceCents = Math.round(priceDollars * 100)

GET /api/admin/products/:id/download (admin-only; streams full PDF)

GET /api/admin/products/:id/open (admin-only; inline PDF preview stream)

Admin UI

On the admin store page, every product row/card has:

Edit (opens existing builder with product loaded)

Price input in dollars (shows $19.99, never cents)

Save

Delete

Download

Open Preview

Acceptance criteria

You can edit title/slug/description and see it update on public page after publish.

You can change price in dollars and checkout uses the updated price.

You can delete a product (it disappears publicly, stays in admin with “deleted” state).

You can download/open the full PDF as admin.

2) Watermarked public preview system (sample pages)
Storage fields

products.pdf_storage_key (private full PDF)

products.preview_storage_key (public preview PDF)

products.preview_page_count (int default 3)

products.preview_updated_at

Admin API

POST /api/admin/products/:id/generate-preview body: { "pageCount": 3 }

Generates a new PDF using first N pages of the full PDF

Applies watermark on every page:

diagonal, repeated or single large

text: NURSENEST PREVIEW • DO NOT DISTRIBUTE

opacity: ~0.12–0.18

GET /api/products/:slug/preview (public; streams preview PDF)

Public product page behavior

Show embedded preview viewer using /api/products/:slug/preview

Do not expose pdf_storage_key or any direct full-PDF URL

Acceptance criteria

Public preview always shows watermark.

Only first N pages are visible.

There is no way to access the full PDF without purchase entitlement.
