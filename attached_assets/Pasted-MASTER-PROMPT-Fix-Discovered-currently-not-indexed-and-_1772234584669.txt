SECTION 1 — DIAGNOSE COMMON INDEXING BLOCKERS
────────────────────────────────────────

A) robots.txt
- Ensure robots.txt exists at /robots.txt and is valid.
- Do NOT disallow /lessons, /blog, /anatomy, /flashcards, /practice if they should be indexable.
- Ensure sitemap is referenced in robots.txt.

B) Meta robots / X-Robots-Tag
- Ensure public pages are not accidentally emitting noindex/nofollow.
- Confirm server does not add X-Robots-Tag: noindex headers on public routes.

C) Canonicals
- Ensure each public page outputs a correct canonical URL.
- Fix any bug where all routes share the same canonical (often happens with SPAs).
- Ensure Google-selected canonical should match user-declared canonical.

D) Status codes and redirects
- Ensure key URLs return HTTP 200.
- Avoid redirect chains and 302s on canonical pages.
- Ensure trailing slash rules are consistent.

E) SPA rendering issues
- Many React SPAs ship an empty HTML shell. If public routes are not server-rendered or pre-rendered, Google may delay crawling/indexing.
- Ensure public SEO pages have crawlable HTML content at initial response or via pre-rendered snapshots.
