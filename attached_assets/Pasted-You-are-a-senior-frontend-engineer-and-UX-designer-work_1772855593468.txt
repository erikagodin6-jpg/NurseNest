You are a senior frontend engineer and UX designer working on the NurseNest exam interface.

Your task is to refine the answer option styling so the test-taking experience feels closer to a professional nursing exam interface while still remaining unique to NurseNest.

The interface must remain legally distinct and use the NurseNest brand system.

Do not copy any proprietary exam UI pixel-for-pixel.

---

GOAL

Improve the answer option design so it feels like a real exam environment instead of a typical web quiz.

The interface should feel:

clean
serious
minimal
highly readable
professional

Avoid designs that feel like mobile app buttons or card tiles.

---

REMOVE CURRENT BOXED ANSWERS

Remove large rounded cards or containers around each answer choice.

Do not place answers inside:

large boxes
cards
pill buttons
shadowed containers

Answer options should appear as simple rows of text.

---

ANSWER OPTION STRUCTURE

Each answer should appear in a vertical list using this structure:

radio button or checkbox marker
answer letter label (A, B, C, D)
answer text

Example layout:

○ A. The patient should receive intravenous fluids.

○ B. The nurse should administer potassium replacement.

○ C. The patient requires oxygen therapy.

○ D. The nurse should notify the physician immediately.

Spacing between answers should be consistent but not excessive.

---

VISUAL STYLE

Answer choices must be:

left aligned
cleanly spaced
easy to scan quickly
high contrast for readability

Use minimal separators if needed.

Avoid heavy borders.

Maintain a calm clinical aesthetic.

---

SELECTION STATE

When a user selects an answer:

the radio button or checkbox becomes filled
the row receives a subtle highlight

The highlight should be very light and neutral.

Avoid bright colors or strong backgrounds.

---

MULTIPLE RESPONSE QUESTIONS

For select-all-that-apply questions:

use checkboxes instead of radio buttons.

Maintain identical spacing and layout.

Example:

☐ A. Increased respiratory rate
☐ B. Hypotension
☐ C. Fever
☐ D. Bradycardia

---

FOCUS AND ACCESSIBILITY

Ensure:

keyboard navigation works
tab focus states are visible
screen readers can identify answer labels

Use semantic HTML where possible.

---

RESPONSIVE DESIGN

The answer layout must work perfectly on:

desktop
tablet
mobile devices

On mobile, maintain comfortable vertical spacing and large tap targets.

---

THEME INTEGRATION

The answer interface must integrate with the NurseNest theme system.

It should inherit typography, spacing, and color tokens from the design system.

The answer rows should not introduce new visual styles outside the brand system.

---

IMPLEMENTATION

Identify the current answer option component used in the exam interface.

Refactor the component so:

answer cards are removed
answers render as simple rows
radio or checkbox controls are used
selection highlighting is subtle

Do not change the question engine logic.

Only update presentation and styling.

---

OUTPUT

Provide:

updated component code
CSS or styling updates
notes explaining how the new layout works
confirmation that the UI remains responsive and accessible
