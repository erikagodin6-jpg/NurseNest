You are working on the **ApplyNest production codebase (Vite + React + Express)**.

Implement a **"Don't show again for today" option on all site popups/modals** so users can temporarily suppress them without permanently disabling important messaging.

This must apply to all current and future popups including:

* newsletter signup prompts
* login prompts
* account creation prompts
* premium upgrade prompts
* resume builder upsells
* scholarship alerts
* feature announcements
* employer onboarding prompts

Do NOT remove any existing popup functionality.
This feature simply allows the user to suppress the popup for **24 hours**.

---

FEATURE REQUIREMENTS

Each popup modal must include a small checkbox near the close button:

□ Don't show again today

When the user checks this box and closes the popup:

1. The popup must not appear again for **24 hours**
2. The preference must persist across:

   * page navigation
   * browser refresh
   * returning to the site later that day

---

IMPLEMENTATION

Use **localStorage** to store suppression timestamps.

When a popup closes:

store:
localStorage.setItem("popup_{popupID}_suppressed_until", timestamp)

timestamp should equal:

current_time + 24 hours

---

POPUP LOAD LOGIC

Before rendering any popup:

1. check localStorage for suppression key
2. compare stored timestamp with current time

If:

current_time < suppressed_until

then:

do NOT render the popup.

If the timestamp is expired or missing:

render popup normally.

---

EXAMPLE REACT UTILITY

Create reusable helper:

/client/utils/popupSuppression.ts

export function shouldShowPopup(popupId) {
const key = `popup_${popupId}_suppressed_until`
const value = localStorage.getItem(key)

if (!value) return true

const now = Date.now()
return now > Number(value)
}

export function suppressPopup(popupId) {
const tomorrow = Date.now() + 24 * 60 * 60 * 1000
localStorage.setItem(`popup_${popupId}_suppressed_until`, tomorrow)
}
