NURSENEST GLOBAL SEO PAGE SYSTEM

Build a complete SEO content architecture for NurseNest that creates indexable pages for every exam, tier, and country while keeping the exam tools behind authentication.

STACK
Frontend: React (Vite)
Backend: Express
Database: existing project database

---

## PRIMARY GOAL

Increase organic traffic by creating high-quality SEO landing pages targeting nursing exam preparation searches.

Each page must:

• rank for exam-specific keywords
• contain educational content
• include sample practice questions
• encourage signups
• link to NurseNest study tools
