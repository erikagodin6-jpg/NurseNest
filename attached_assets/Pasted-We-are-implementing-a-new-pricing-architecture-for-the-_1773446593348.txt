This system must work with Stripe subscriptions and one-time purchases.

- NP
- Allied Health

Each tier must have its own pricing plans and access control.

2. Admin must be able to:

Subscription plans must create Stripe subscriptions.

Lifetime plans must create Stripe one-time purchases.

Users only access the tier they purchased.
