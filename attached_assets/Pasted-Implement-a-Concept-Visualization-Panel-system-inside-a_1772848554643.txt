Implement a "Concept Visualization Panel" system inside all NurseNest question explanations.

The goal is to improve learning retention and make explanations more visually intuitive.

------------------------------------------------

FEATURE 1 — VISUAL CONCEPT PANEL

Every question explanation must include a small visual diagram illustrating the concept.

Examples of diagrams:

pathophysiology flow diagrams
organ system diagrams
lab interpretation charts
clinical decision trees

The diagram should appear directly below the explanation text.

------------------------------------------------

FEATURE 2 — AUTOMATIC VISUAL GENERATION

When questions are created, generate a simple visual concept diagram summarizing the key mechanism being tested.

The diagram must:

use minimal text
show directional flow
highlight cause-and-effect relationships
focus on the key mechanism tested in the question

------------------------------------------------

FEATURE 3 — DIAGRAM STYLE

Use NurseNest design tokens.

Colors:
soft lavender
sage green
pale blue
soft coral accents

Design style:

minimalist
educational
clean vector style
soft rounded shapes
clear arrows
high readability

Avoid complex medical illustrations.

These diagrams should be simple teaching visuals.

------------------------------------------------

FEATURE 4 — VISUAL TYPES

Support four diagram types:

Pathophysiology flow diagrams
Example:

insulin deficiency
↓
lipolysis
↓
ketone production
↓
metabolic acidosis

Organ diagrams
Example:
kidney nephron highlighting sodium reabsorption.

Lab interpretation flowcharts.

Clinical decision trees.

------------------------------------------------

FEATURE 5 — MOBILE RESPONSIVE

Ensure diagrams resize cleanly for mobile screens.

Use stacked layouts if necessary.

------------------------------------------------

FEATURE 6 — EXPLANATION STRUCTURE

All question explanations should follow this format:

Correct Answer

Short Explanation

Visual Concept Panel

Clinical Pearl

Exam Trap

------------------------------------------------

FEATURE 7 — REUSABLE COMPONENT

Create a reusable React component called:

ConceptVisualizationPanel

Props should include:

diagram type
node labels
arrow relationships
highlighted nodes

------------------------------------------------

FEATURE 8 — PERFORMANCE

Ensure diagrams render as lightweight SVG graphics.

Avoid large images.

------------------------------------------------

FINAL RESULT

Students should be able to quickly understand complex nursing concepts through simple visual diagrams embedded directly in question explanations.
