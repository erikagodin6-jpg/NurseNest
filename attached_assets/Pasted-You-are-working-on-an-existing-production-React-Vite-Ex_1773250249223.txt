BREADCRUMB IMPLEMENTATION REQUIREMENTS

Create or refactor a shared breadcrumb structured data system so every eligible page can output valid JSON-LD consistently.

Use Schema.org `BreadcrumbList` JSON-LD.

Each breadcrumb trail should:
- use `@context`: `https://schema.org`
- use `@type`: `BreadcrumbList`
- include `itemListElement`
- use `ListItem` for each step
- include `position`
- include `name`
- include `item` URL for linked levels where appropriate

Make sure breadcrumb trails reflect the actual site hierarchy.

Examples of desired hierarchy patterns:
- Home
- Home > Lessons
- Home > Lessons > [Lesson Category]
- Home > Lessons > [Lesson Category] > [Lesson Title]
- Home > Exams > [Exam Type]
- Home > Flashcards > [Category]
- Home > Store > [Product Category] > [Product Name]

Only include breadcrumb markup on pages where it makes sense and where the page is indexable.
