MASTER PROMPT: Build SEO → Free Practice → Subscription Conversion Funnel for Test Mode

You are a senior full-stack engineer working inside my existing NurseNest codebase (Vite + React frontend, Express backend, existing subscription system, flashcards, AI verification, study modes).

Your goal is to implement an SEO-driven acquisition and conversion system that:

1) Ranks for nursing exam practice keywords

3) Converts users into paid subscriptions for full test mode access

────────────────────────────────────────
SECTION 1 — SEO PRACTICE LANDING PAGE FRAMEWORK
────────────────────────────────────────

Create a reusable dynamic page template:

Route:
  /practice/[exam-type]/[topic-slug]

Examples:
  /practice/nclex/hyperkalemia
  /practice/rexpn/prioritization
  /practice/np/dka
  /practice/nclex-nextgen/cardiac-case-study

Each practice page must contain:

1) SEO-optimized long-form content (1500–3000 words)
   - H1 targeting primary keyword
   - H2/H3 structured headings
   - Internal links to related lessons
   - FAQ section
   - Schema markup (FAQ + Article)

2) Embedded interactive practice questions (limited free access)
   - Pull 5–10 sample questions from the test bank
   - Question types: multiple choice minimum
   - Optional: SATA and case-based if supported

3) Performance feedback block
   - Score summary
   - “Passing likelihood indicator”
   - Visual progress bar
   - Weak topic suggestions

4) Conversion block
   - CTA section:
     “Unlock 300+ practice questions”
     “Access full adaptive mock exams”
     “Track your performance analytics”
   - Subscription plan comparison
   - Upgrade button integrated with existing billing

IMPORTANT:
Only allow 5–10 free questions per topic per day for non-subscribers.
Subscribers get unlimited practice.

SECTION 2 — FREE LIMIT ENFORCEMENT
────────────────────────────────────────

Implement server-side limits:

Free users:
- 5–10 questions per topic per day
- Limited performance breakdown
- No adaptive exam simulation
- No detailed analytics

Paid users:
- Unlimited questions
- Adaptive exam mode
- Detailed analytics
- Save exam history
- Performance prediction tools

Add:
POST /api/practice/session
POST /api/practice/submit-answer
GET  /api/practice/results

Track:
- daily_question_attempt_count per user per topic
- enforce limit server-side

Return error:
"FREE_LIMIT_REACHED"

Trigger upgrade modal automatically.
