The system must enforce:

3) Required calculation minimums (where appropriate)
4) Allowed question format constraints
5) Publishing validation rules
6) Scaled scoring simulation where applicable
7) Region toggles (Canada vs US where relevant)

STEP 3 — ALLOWED FORMAT MATRIX
---------------------------------------------------------

Create a format matrix:

Common allowed:
- MCQ_SINGLE
- CASE_BASED_CLUSTER
- PRIORITIZATION

Conditionally allowed:
- NUMERIC_ENTRY (MLT, Pharm Tech, Paramedic, RRT, Imaging)
- IMAGE_INTERPRETATION (Imaging, PT optional)
- DIFFERENTIAL_DIAGNOSIS (PT)
- SIMULATION_TREE (OT US only)

Prohibited globally:
- NGN Bowtie
- Drag-drop cloze
- Highlight text
- Matrix multi unless profession uses it

---------------------------------------------------------
STEP 4 — MOCK EXAM ASSEMBLY ALGORITHM
---------------------------------------------------------

When generating a mock exam:

1) Allocate questions per domain using weight range midpoint.
2) Reserve required minimum format slots.
3) Inject required case clusters first.
4) Insert calculation minimums (if required).
5) Fill remaining with allowed formats.
6) Verify no domain exceeds max weight.
7) Verify no prohibited formats used.
8) Verify difficulty distribution:
   - 30% moderate
   - 40% applied reasoning
   - 30% complex clinical reasoning

---------------------------------------------------------
STEP 5 — VALIDATION BEFORE PUBLISHING
---------------------------------------------------------

Block publishing if:
- Domain outside ±3%
- Case minimum not met
- Calculation minimum not met (if required)
- Prohibited format detected
- >20% recall-only items
- Duplicate or near-duplicate stems

---------------------------------------------------------
STEP 6 — SCORING
---------------------------------------------------------

If scoringModel = dichotomous:
  1 point per item

If scaled:
  Convert raw to scaled (200–800 or 100–600 depending on profession)
  Provide:
    - Estimated pass probability
    - Domain performance breakdown
    - Remediation recommendations

Never claim guaranteed pass.
