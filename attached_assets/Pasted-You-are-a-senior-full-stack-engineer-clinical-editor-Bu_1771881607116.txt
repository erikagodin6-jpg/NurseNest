GOAL

1) Displays common lab reference ranges with BOTH SI and conventional units (Canada + USA)
2) Lets the user toggle:
   - Show SI only
   - Show conventional only
   - Show both
3) Is optimized for printing (one-click “Print / Save as PDF”)
4) Includes a “Quick Interpretation” column (high-yield nursing implications)

- For each lab, include: Test name, SI unit + range, conventional unit + range, brief interpretation pearls.
