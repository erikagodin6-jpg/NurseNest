3) Make the AI prompt strict + add JSON validation + retry
Strict JSON prompt (must be used)
Your AI call should demand only JSON and explicitly include the schema.
Key behaviors:
“Return JSON only. No markdown.”
“Do not include trailing commas.”
“Do not include keys not in schema.”
Parse/validate/retry logic
Do:
call AI
JSON.parse
schema validate (zod ideal)
on fail: retry once with “Fix JSON” prompt including the raw output
Pseudo:
async function getStructuredContent(prompt: string) {
  const res1 = await runAi(prompt);
  const parsed1 = safeParse(res1);
  if (parsed1.ok) return parsed1.value;

  const fixPrompt = buildFixJsonPrompt(res1.raw, prompt);
  const res2 = await runAi(fixPrompt);
  const parsed2 = safeParse(res2);
  if (parsed2.ok) return parsed2.value;

  throw new Error("AI_JSON_PARSE_FAILED");
}
This one change eliminates most “AI did something weird” failures.
