I need to redesign the entire exam generation and delivery system so that every user tier gets its own distinct question bank, exam style, difficulty profile, and clinical reasoning depth.

Current problem:
The platform is not sufficiently separating RPN/LVN, RN, and NP exam content. I need strict tier-based differentiation so that:
- RPN/LVN users get RPN/LVN-level questions only
- RN users get RN-level questions only
- NP users get NP-level questions only
- Different tiers must NOT share the same question wording, same rationales, same clinical complexity, or same distractor style unless explicitly configured for review products
- Difficulty must scale appropriately by tier
- The experience must feel like three distinct exam ecosystems, not one bank with labels

GOAL:
Build a robust multi-tier exam system that supports:
1. Separate exam blueprints by tier
2. Separate question difficulty models by tier
3. Separate cognitive depth by tier
4. Separate distractor generation by tier
5. Separate rationale style by tier
6. Separate UI labels and metadata by tier
7. Separate backend filtering and authorization by tier
8. Optional admin controls for overlap rules

PART 2 — TIER DIFFERENTIATION RULES
==================================================

Implement hard-coded content standards for each tier.

RPN/LVN tier:
- Focus on foundational nursing knowledge
- Emphasize stable or predictable patient scenarios
- Prioritize direct recognition, safe intervention, focused assessment, basic pharmacology, and common disease management
- Questions should test safe practical nursing judgment, prioritization within predictable settings, and recognition of when to escalate
- Distractors should be plausible but somewhat more concrete and less abstract
- Stem length should generally be shorter to moderate
- Case complexity should be lower
- Lab interpretation should stay more straightforward
- Pharmacology should focus more on common medications, adverse effects, contraindications, monitoring, and nursing implications
- SATA or matrix style can exist if enabled, but standard multiple choice should remain common
- Difficulty mix recommendation:
  easy 20%
  moderate 50%
  hard 25%
  very_hard 5%

RN tier:
- Focus on broader clinical judgment, prioritization, delegation, unstable patient recognition, higher complexity assessment, pathophysiology integration, and more nuanced nursing interventions
- Include more layered scenarios than RPN
- Increase ambiguity between answer choices
- Require stronger interpretation of labs, symptoms, trends, and safest next action
- Include multi-step reasoning and prioritization
- Distractors should be intentionally close and realistic
- Stem length moderate to long
- More scenario-based questions
- Difficulty mix recommendation:
  easy 10%
  moderate 40%
  hard 35%
  very_hard 15%

NP tier:
- Focus on advanced assessment, differential diagnosis, diagnostic reasoning, evidence-based management, pharmacotherapeutics, interpretation of diagnostics, and advanced clinical decision-making
- Questions must be clinically dense and graduate level
- Distractors should reflect errors in diagnostic reasoning, inappropriate workups, incorrect prescribing, incomplete management, or missed red flags
- Use more advanced case vignettes
- Require synthesis rather than recall
- Include nuanced patient factors: age, comorbidities, contraindications, sequencing, follow-up, and guideline-informed decisions
- Difficulty mix recommendation:
  easy 5%
  moderate 25%
  hard 40%
  very_hard 30%

Admin:
- Can preview all tiers
- Can filter by tier
- Can manually cross-list specific content if desired

PART 3 — QUESTION GENERATION RULES
==================================================

When generating questions, use tier-aware generation prompts.

RPN question generation rules:
- Use simpler clinical layers
- Emphasize safe nursing actions
- Prefer one best answer based on foundational knowledge
- Avoid excessive diagnostic ambiguity
- Focus on recognition, action, monitoring, teaching, and escalation

RN question generation rules:
- Use richer scenarios
- Require prioritization and more advanced interpretation
- Include unstable trends, competing priorities, and delegation/scope considerations
- Increase similarity between distractors
- Test clinical judgment more heavily than recall

NP question generation rules:
- Use advanced clinical language
- Require diagnosis, workup, prescribing judgment, contraindication awareness, and follow-up planning
- Include guideline-consistent management logic where appropriate
- Test clinical reasoning depth, not memorization alone
- Rationales should discuss why other options are wrong at an advanced level

Build separate generator functions:
- generateRpnQuestion()
- generateRnQuestion()
- generateNpQuestion()

Do not use one generic generator with superficial labels.

PART 4 — DIFFICULTY ENGINE
==================================================

Create a difficulty engine that is tier-aware.

Difficulty must not mean only “harder facts.”
Difficulty should be based on:
- ambiguity of distractors
- number of reasoning steps
- severity and instability of scenario
- amount of data interpretation required
- complexity of prioritization
- overlap of plausible answer choices
- pharmacology complexity
- diagnostic complexity
- guideline nuance for NP tier

Implement a scoring model:

difficultyScore =
  scenarioComplexity +
  distractorCloseness +
  clinicalJudgmentDepth +
  interpretationLoad +
  prioritizationBurden +
  pharmacologyComplexity +
  diagnosticNuance

Then map score ranges to:
- easy
- moderate
- hard
- very_hard

Make score thresholds configurable by tier because a “hard RPN” question is not the same as a “hard NP” question.

==================================================

Rationales must differ by tier as well.

RPN rationale style:
- concise but educational
- focus on safety, recognition, and nursing action
- explain why the best answer is safest or most appropriate
- briefly explain major distractors

RN rationale style:
- moderate to detailed
- emphasize prioritization, patient safety, interpretation, and nursing judgment

- include subtle reasoning where needed

NP rationale style:
- detailed and advanced

- discuss why the correct answer fits best clinically
- explain contraindications, guideline reasoning, alternative management, or why distractors fail logically
