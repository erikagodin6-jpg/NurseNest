MASTER IMPLEMENTATION PROMPT — NURSENEST SEO + VISUAL TRAFFIC ENGINE

You are a senior full-stack engineer building a production feature inside the NurseNest platform.

The system must automate SEO content production, infographic generation, Pinterest pins, internal linking, and publishing workflows.

This tool will be called the:

NurseNest SEO + Visual Traffic Engine

The system must integrate with an existing Node.js / Express backend and React frontend.

The purpose of the system is to allow an admin to enter a keyword and automatically generate:

• topic clusters
• pillar + supporting articles
• branded infographics
• Pinterest pins
• internal linking maps
• a publish queue

All content must be optimized for nursing education and exam preparation.

---

PART 1 — DATABASE SCHEMA

Implement the following PostgreSQL tables.

Tables required:

seo_clusters
seo_articles
infographic_templates
seo_infographics
seo_pins
seo_internal_links
publish_queue
qc_runs

Each table must contain fields necessary for:

cluster metadata
article metadata
image asset tracking
pin asset tracking
internal linking
quality control checks
publish scheduling

Use UUID primary keys.

Implement indexes for status fields and foreign key relationships.
