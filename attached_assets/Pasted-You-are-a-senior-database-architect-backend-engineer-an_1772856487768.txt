You are a senior database architect, backend engineer, and assessment platform engineer.

Design a production-ready PostgreSQL database architecture for the NurseNest nursing exam preparation platform.

The platform must support very large educational datasets and scale to at least 50,000+ exam questions without performance degradation.

The system must support the following features:

lessons
question bank
flashcards
clinical case studies
adaptive exams
progress tracking
analytics dashboards
subscription access control
content protection
admin content management

---

SYSTEM ARCHITECTURE REQUIREMENTS

The architecture must separate the platform into three logical layers:

Content Layer
Stores educational assets.

Delivery Layer
Handles exam sessions, practice quizzes, and question delivery.

Analytics Layer
Handles user performance data, mastery scoring, and dashboards.

Ensure the architecture keeps these layers loosely coupled.

---

DATABASE TECHNOLOGY

Use PostgreSQL as the core relational database.

Design tables with:

proper indexing
normalized relationships
JSON support where appropriate
scalability for large datasets
