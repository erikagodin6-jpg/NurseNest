Step 2: Add Page-Length Controlled AI Generation
Right now AI probably just dumps text.
You want AI to generate content based on page count target.
That requires:
Estimated characters per page
Section distribution logic
Prompt to Add Page Length Control
Implement page-length controlled AI generation.

1) Each template should define:
   - Estimated characters per page (based on font size and margins)
   - Section weight distribution (e.g., Pathophysiology 20%, Questions 30%, etc.)

2) When user selects page length target (e.g., 45 pages):
   - System calculates total character target
   - Distributes character budget across sections
   - AI generates content to meet approximate character count

3) After generation:
   - Auto-paginate content into pages
   - If overflow occurs, create additional pages
   - If underflow occurs, expand summary or add review questions

4) Never require manual resizing or manual page splitting.
