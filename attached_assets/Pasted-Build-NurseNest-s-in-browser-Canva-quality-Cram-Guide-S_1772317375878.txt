Build NurseNest’s in-browser Canva-quality Cram Guide Studio that lets Erika create premium-looking multi-page PDFs, generate/edit images, and sell them with Stripe + PayPal, including watermarked public previews and secure downloads.

PRIMARY UX GOAL
Erika can do everything from her browser at home: create, design, generate images, preview, price, publish, edit, delete, and download products.

A) CREATE “CRAM GUIDE STUDIO” (ADMIN)
Route: /admin/studio (and /admin/studio/:productId)

Studio layout (3 panels):
1) LEFT: Blocks + Templates + Assets
   - Templates tab: load presets (10+)
   - Blocks tab: drag/click to add blocks
   - Assets tab: image library + icons + uploaded PDFs thumbnails
2) CENTER: Canvas
   - multi-page scroll
   - A4/Letter toggle
   - zoom
   - reorder pages
   - duplicate page
   - snap grid (simple)
3) RIGHT: Style + Settings
   - theme preset selector (Soft Pastel default, Clinical Minimal, Bold Study, High Contrast)
   - palette lock (brand colors only)
   - typography preset (font pairings)
   - spacing controls
   - header/footer controls + logo placement
   - watermark controls (preview only)
   - export controls (PDF quality)

B) “CANVA-LITE” BLOCK SYSTEM (MUST LOOK PREMIUM)
Blocks (minimum):
- Title hero (with soft background card)
- Section header
- Bullet list card
- Two-column card
- Clinical pearls callout
- Red flags callout
- Exam traps callout
- Labs table (styled)
- Med table (styled)
- Algorithm steps block
- Mnemonic block
- Image block (with rounded corners + caption)
- Checklist block
- Quick-reference grid (2x2, 3x3)
- Footer block (auto on every page)

Rules:
- Default rounded corners 16px
- Soft shadow
- Plenty of whitespace (auto margins)
- No harsh black lines
- Consistent typography hierarchy

D) AI CONTENT GENERATOR (STRUCTURED JSON, NOT PROSE)
Inside Studio add:
- Topic input
- Exam target dropdown (REx-PN / NCLEX-PN / NCLEX-RN / NP)
- Region toggle (Canada vs USA variants)
- Depth toggle (1-page cheat / quick guide / full guide / bundle)
- Button: “Generate Content”

AI must return strict JSON:
{
  title, subtitle,
  sections:[{heading, bullets[], pearls[], redFlags[], examTraps[]}],
  tables:[{type:'labs'|'meds'|'compare', headers[], rows[]}],
  algorithms:[{title, steps[]}],
  mnemonics:[],
  practiceQuestions:[{stem, options[], answerIndex, rationale}]
}

Then add:
- Button “Auto Layout”
  -> picks a template
  -> injects content into blocks
  -> creates a final 1-page cheat sheet page automatically

E) IMAGE GENERATOR (LONG PROMPTS + EDITING + TEXT-FREE MODE)
Add Studio tab: “Image Lab”
Features:
- Prompt textarea (large, supports long prompts)
- Negative prompt textarea
- Size/aspect presets (A4 header banner, square, 4:5, 16:9)
- Generate 1–4 variations
- Simple edits: crop/rotate, background remove, brightness/contrast, recolor tint, blur, vignette, add shape overlays
- “Insert to Canvas” button

Text-free requirement:
- Provide a “Text-Free Mode” toggle that:
  - injects strong negative prompt: “no text, no letters, no words, no numbers, no watermark, no labels”
  - blocks any user instruction requesting text overlay
  - after generation, run a basic text-detection check and warn if text is detected; allow “Regenerate” button
(Note: detection can be best-effort; warn not guarantee.)
