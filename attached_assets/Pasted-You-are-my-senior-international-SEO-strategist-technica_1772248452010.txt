en, fr, es, fil, hi, ar, fa, zh-Hans, ja, ko, ur, pa, pt, vi, ht

====================================================
PART 1 — LANGUAGE ROI ANALYSIS ENGINE
====================================================

Create a language priority scoring model.

For each language, score based on:

1) Global nursing population relevance
2) Immigration patterns to US/Canada
3) NCLEX search demand (estimate using keyword API if available)
4) Competition strength in that language
5) Monetization potential (card usage, Stripe compatibility, etc.)
6) Content production difficulty

Score each factor 1–5.

Compute:
ROI_score = weighted sum.

Output ranked list:
- Tier 1 (High ROI)
- Tier 2 (Medium ROI)
- Tier 3 (Long-term expansion)

Store in:
language_priority table.
