Implement a beta access code system for NurseNest so selected users can test the platform for free.

STACK
Frontend: React (Vite)
Backend: Express
Database: existing project database

GOAL
Allow admins to generate special access codes that grant free premium access for a limited time.

---

## DATABASE STRUCTURE

Create a new table or collection called:

beta_codes

Fields:

id
code
createdAt
expiresAt
maxUses
uses
planGranted
durationDays
active

Example record:

{
code: "NURSENEST-BETA-2026",
planGranted: "premium",
durationDays: 30,
maxUses: 50,
uses: 0,
active: true
}

---

## CODE GENERATION

Create a function that generates secure codes.

Example format:

NN-BETA-XXXXXX

Example code generator:

function generateBetaCode() {
const chars = "ABCDEFGHJKLMNPQRSTUVWXYZ23456789"
let result = "NN-BETA-"

for (let i = 0; i < 6; i++) {
result += chars[Math.floor(Math.random() * chars.length)]
}

return result
}

---

## ADMIN PANEL

Add a section in the admin dashboard:

"BETA ACCESS CODES"

Admin should be able to:

• generate new codes
• set expiration date
• set max uses
• set duration of free access
• deactivate codes

---

## REDEEM PAGE

Create a page:

/redeem-code

User enters code.

Example input field:

Enter Access Code

---

## BACKEND VALIDATION

When a user submits a code:

1. Find code in database
2. Check if active
3. Check expiration date
4. Check uses < maxUses

If valid:

• increase uses count
• grant user premium access
• set expiration date on account

Example:

user.plan = "premium"
user.planExpires = today + durationDays

---

## ERROR HANDLING

Return clear messages:

Invalid code
Code expired
Code already used
Code limit reached

---

## SUCCESS MESSAGE

Show:

"Access granted! You now have free premium access."

---

## SECURITY

Ensure:

• codes cannot be reused beyond maxUses
• codes cannot be brute forced easily
• redemption requires login

---

## OPTIONAL FEATURE

Allow codes to be tied to specific tiers.

Example:

RPN testers
RN testers
NP testers

---

## END RESULT

Admins can generate invite codes and give them to testers.

Users enter the code once and automatically receive free premium access for a limited time.
