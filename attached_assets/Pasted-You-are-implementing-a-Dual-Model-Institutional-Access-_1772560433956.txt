SECTION 2 — LICENSE MODEL LOGIC
==================================================

COHORT MODEL:
- All student accessEnd = institution.semesterEndDate
- Seat cap enforced
- Bulk enrollment via CSV supported
- Enrollment blocked after semesterEndDate

INDIVIDUAL MODEL:
- Each student accessEnd = 
    if examDate provided → examDate
    else now + defaultDurationDays
- Rolling enrollment
- Seat cap enforced
- Independent expiration per student
