You are a senior frontend security engineer working on the NurseNest educational platform.

Your task is to implement strong content protection measures to discourage users from copying, exporting, or screenshotting proprietary educational content.

The goal is to protect the NurseNest question bank, lessons, and exam content while maintaining a smooth user experience.

It is understood that no browser-based system can completely stop screenshots, but we must implement strong deterrents.

---

PRIMARY GOALS

Prevent or discourage:

text copying
copy/paste actions
right-click copying
screen capture shortcuts
developer tool scraping

These protections should apply to:

lesson content
exam questions
flashcards
case studies
question banks

---

TEXT SELECTION BLOCKING

Disable text selection on protected content.

Apply CSS rules:

user-select: none

Ensure this applies to:

questions
answers
lesson text
flashcard content

---

COPY / PASTE PREVENTION

Block common keyboard shortcuts including:

Ctrl + C
Ctrl + V
Ctrl + X
Cmd + C
Cmd + V
Cmd + X

Intercept these events using JavaScript and prevent default behavior.

---

RIGHT CLICK PROTECTION

Disable right-click context menus on protected content.

Prevent actions like:

copy
inspect element
save text

---

SCREENSHOT SHORTCUT DETECTION

Attempt to detect common screenshot shortcuts.

Examples:

Print Screen
Cmd + Shift + 3
Cmd + Shift + 4
Windows Snipping Tool shortcuts

When detected:

show a warning overlay stating:

"Screenshots are disabled to protect NurseNest educational content."

Note: Detection should be best-effort and non-intrusive.

---

DEV TOOLS DETECTION

Detect when browser developer tools are opened.

Possible signals:

window size anomalies
debugger detection
performance checks

If developer tools are detected:

blur protected content
display a warning overlay

---

WATERMARKING

Add dynamic watermark overlays to protected content.

Watermarks should include:

user email
user ID
timestamp

Example watermark:

"User: [john@email.com](mailto:john@email.com) | NurseNest | 2026"

The watermark should be semi-transparent and repeated across the screen.

This discourages screenshot sharing.

---

TAB SWITCH DETECTION

If the user switches tabs or minimizes the browser during exams:

pause the exam
blur exam content

Display message:

"Exam paused because the window lost focus."

---

CONTENT OBFUSCATION

Prevent easy scraping of content by:

avoiding plain text rendering in raw HTML
obfuscating sensitive content where possible
rendering questions dynamically from the API

---
