Extend the NurseNest Hybrid AI Content Engine so it can generate high-quality content for ALL allied health pages and ALL new grad pages, including exam-style questions, realistic CAT-style questions where appropriate, and flashcards that meet strict quality, variety, and formatting standards.

The system must support large-scale generation for:
- nursing tiers
- allied health categories
- specialty certification pages
- new grad pages
- transition-to-practice content
- clinical orientation and early-career readiness content

CONTENT TYPES
For every allied health and new grad content area, support:
1. exam-style questions
2. CAT-style question pools where applicable
3. flashcards
4. lesson/topic support
5. rationale-linked learning references
6. image suggestion metadata

QUESTION QUALITY STANDARD
Questions must feel like premium exam prep material, not generic trivia.
Every generated question must be:
- clinically realistic
- professionally written
- grammatically polished
- relevant to real practice
- tier and specialty appropriate
- varied in difficulty
- free of obvious distractors
- free of repeated phrasing patterns
- educationally useful

QUESTION VARIETY REQUIREMENT
The engine must generate a balanced mix of question types. Do not generate the same style repeatedly.

Supported formats must include:
- standard single-best-answer questions
- clinical scenario questions
- prioritization questions
- delegation questions
- patient teaching questions
- assessment-first questions
- safety questions
- therapeutic communication questions
- next-best-action questions
- select-all-that-apply where appropriate
- NGN/CAT-style clinical judgment questions where supported by the page/exam type
- unfolding case style items where appropriate
- recognition and cue-identification items
- condition differentiation items
- emergency/escalation items

CAT / REALISM REQUIREMENT
For CAT-style banks and adaptive-style pools:
- questions must vary naturally in difficulty
- difficulty progression must not be random; it must be tagged and structured
- include easy, moderate, hard, and high-clinical-judgment items
- difficult items must reflect ambiguity and prioritization, not just obscure facts
- easy items should still be clinically sound and useful
- realistic case-based items should mirror actual patient care situations
- avoid “textbook obvious” cues that make the answer too easy
- distractors must be plausible within scope of practice

QUESTION DISTRIBUTION RULES
For each batch, distribute questions across:
- recall/knowledge
- comprehension
- application
- analysis/clinical judgment
- prioritization/delegation
- safety/risk reduction

Add configurable percentages by exam type and specialty.

ALLIED HEALTH RULES
All allied health categories must have their own generation profiles.
For each allied health specialty, store:
- scope of practice
- learner level
- certification/exam style if applicable
- common task domains
- topic blueprints
- appropriate terminology
- prohibited out-of-scope content

Do not generate nursing-specific scope language for non-nursing allied health pages unless explicitly intended.
Do not blur respiratory therapy, paramedic, emergency nursing, and general nursing content into one style.
Each specialty must feel distinct.

NEW GRAD PAGE RULES
New grad pages must focus on:
- transition to practice
- first-year nurse readiness
- shift organization
- prioritization
- delegation
- communication
- documentation
- handling deterioration
- medication safety
- asking for help/escalation
- workload management
- confidence building through realistic practice scenarios

New grad questions should emphasize real-world floor/acute-care decision making, not only textbook disease facts.

FLASHCARD REQUIREMENTS
Flashcards must follow strict formatting and quality rules.

Every flashcard must include:
- concise front
- precise back
- optional explanation or rationale
- linked topic/category
- linked lesson slug where available
- image suggestion metadata if appropriate

Flashcard rules:
- one high-yield concept per card
- no bloated paragraphs
- no vague wording
- no filler
- back text must be concise but educational
- flashcards should support spaced repetition and rapid review
- vary card style across:
  - definition
  - recognition
  - intervention
  - lab/diagnostic clue
  - adverse effect
  - prioritization pearl
  - patient teaching pearl
  - red flag
  - scope reminder
  - clinical judgment cue

CONTENT LENGTH CONTROLS
Add strict configurable content-length validators.

QUESTION STEM LENGTH
- short factual questions: limited concise length
- standard scenario questions: medium length
- advanced case questions: longer but still readable
- reject unnecessarily bloated stems

RATIONALE LENGTH
- must be long enough to teach
- must explain why the correct answer is correct
- must explain why distractors are wrong except where CAT exam logic excludes this
- must remain concise and not become textbook chapters

FLASHCARD LENGTH
- front must be short
- back must be concise and high-yield
- explanation must not exceed configured maximum
- reject long rambling outputs

LESSON SUPPORT LENGTH
- summaries must be compact
- key points must be scannable
- exam tips must be direct and useful

QUALITY CONTROLS
Add validation checks for:
- readability
- clinical plausibility
- distractor plausibility
- repeated wording across batch
- answer position bias
- overuse of the same correct answer pattern
- repeated rationale templates
- excessive verbosity
- triviality
- poor specialty fit
- poor new-grad relevance
- weak educational value

If a question is too easy, too vague, too repetitive, too unrealistic, or too generic, flag it for revision.

BLUEPRINT / COVERAGE RULES
For every allied health and new grad page, create generation blueprints that define:
- domains
- subtopics
- skills tested
- question mix
- flashcard mix
- realism level
- patient population
- practice setting
- scope boundaries

Require coverage balancing so one batch does not overproduce the same subtopic repeatedly.

IMAGE USE RULES
For rationales and flashcards:
- suggest existing approved images only
- only attach images when clinically relevant
- avoid decorative-only image matching
- prioritize realistic, educational, polished images
- no cartoonish or childish visuals
- score image relevance by topic and setting
