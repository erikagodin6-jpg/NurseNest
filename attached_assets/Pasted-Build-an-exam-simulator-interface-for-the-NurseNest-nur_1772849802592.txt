Build an exam simulator interface for the NurseNest nursing study platform.

The interface should feel similar to real nursing licensing exams such as NCLEX and REx-PN but must be an original design.

Do not copy Pearson VUE layouts exactly.

The UI should simulate the experience of taking a nursing exam.

------------------------------------------------

EXAM INTERFACE STRUCTURE

Display only one question at a time.

The layout must include:

top navigation bar
main question panel
answer choices
bottom navigation controls
optional tool panel

------------------------------------------------

TOP BAR

Display:

exam timer
question number
flag for review button
progress indicator

------------------------------------------------

QUESTION PANEL

Display:

patient scenario
vital signs
lab values
question prompt

Use a clean readable layout.

------------------------------------------------

ANSWER SECTION

Display answer choices as selectable cards.

Support question types:

single best answer
select all that apply
drag and drop
matrix
calculation

------------------------------------------------

NAVIGATION CONTROLS

Include buttons:

previous question
next question
submit answer

------------------------------------------------

TOOLS

Provide optional tools such as:

highlight text
strikeout answer choices
calculator
notes panel

------------------------------------------------

MODES

Implement two modes:

Tutor Mode
show explanation after each question

Exam Mode
show explanations only after the exam ends

------------------------------------------------

EXPLANATION STRUCTURE

After answering show:

correct answer
explanation
why other options are incorrect
clinical pearl
exam trap

------------------------------------------------

DESIGN

Maintain NurseNest branding:

soft pastel colors
clean typography
rounded UI
minimal distractions
high readability

The interface should feel like a professional medical exam simulator.

------------------------------------------------

MOBILE

Ensure the interface works smoothly on mobile devices.

Large tap targets
sticky navigation
simple scrolling
