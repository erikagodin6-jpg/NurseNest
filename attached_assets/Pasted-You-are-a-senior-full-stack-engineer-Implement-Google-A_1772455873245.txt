PART 2 — SPA Route Change Tracking (CRITICAL)

Because this is a React SPA using Wouter, default pageview tracking will fail after first load.

Create a component called AnalyticsTracker.tsx.

Implementation:

import { useEffect } from "react";
import { useLocation } from "wouter";

export default function AnalyticsTracker() {
  const [location] = useLocation();

  useEffect(() => {
    if (typeof window !== "undefined" && window.gtag) {
      window.gtag("event", "page_view", {
        page_path: location,
        page_location: window.location.href,
        page_title: document.title,
      });
    }
  }, [location]);

  return null;
}

Mount <AnalyticsTracker /> once inside the main App component so it runs globally.
