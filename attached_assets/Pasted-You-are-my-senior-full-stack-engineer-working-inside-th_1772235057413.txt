You are my senior full-stack engineer working inside the existing NurseNest architecture (Vite + React + Express + database + performance tracking + question bank + flashcards + mock exams + study plan system).

I want to build an INTELLIGENT PERFORMANCE-BASED COURSE RECOMMENDATION ENGINE that automatically recommends and LINKS users to specific courses after they complete:

- Flashcard sessions
- Question bank quizzes
- Case sets
- Mock exams (fixed length or CAT)

This must feel clinically intelligent, exam-focused, and personalized — not generic.

====================================================
OBJECTIVE
====================================================

After each completed session, the system must:

1. Analyze performance in depth
2. Identify measurable weaknesses
3. Select the most relevant courses
4. Display clickable course links
5. Allow one-click add to study plan
6. Log recommendation acceptance + improvement over time

====================================================
PART 1 — PERFORMANCE ANALYSIS ENGINE
====================================================

After session completion calculate:

- Overall score
- Score by:
  - Body system
  - Topic
  - Subtopic
  - Question type
  - Clinical priority category (ABCs, safety, pharm, delegation, etc.)
  - Client needs category (if mapped)
- Difficulty-weighted performance
- Time per question
- Confidence rating (if available)
- Missed concept frequency

Create a Weakness Index formula:

Weakness Index =
(Low Accuracy Score × Exam Weight Multiplier × Recency Factor × Difficulty Factor)

Categorize:
- Critical Weakness (Red)
- Moderate Weakness (Yellow)
- Stable (Green)

Identify:
- Top 3 weak domains
- Top 2 weak question types
- Top recurring conceptual errors

PART 3 — SMART RECOMMENDATION LOGIC
====================================================

After analysis:

Select:

PRIMARY RECOMMENDATIONS (1–2)

- High exam weight topics
- Not recently completed
- Within appropriate difficulty range

SECONDARY RECOMMENDATIONS (3–5)

- Question type mastery modules (e.g., SATA mastery)
- Reinforcement content

ADVANCED TRACK (if user >85%):
- Higher difficulty case simulations
- Full-length mock exam
- Adaptive challenge set

FOUNDATIONAL TRACK (if user <60%):
- Core concept rebuild modules
- Lower difficulty review set
- Flashcard reinforcement bundle
